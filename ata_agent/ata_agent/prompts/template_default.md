# Template de referência — Ata de reunião interna

Use como guia de secções (adaptar ao conteúdo real):

- **Cabeçalho:** data, hora (se conhecida), tipo de reunião, participantes.
- **Objetivo / contexto** (opcional, 2–4 linhas).
- **Tópicos discutidos** (lista ou subtítulos).
- **Decisões tomadas** (D-001, D-002, …).
- **Encaminhamentos** (cada item com responsável `[[Nome]]` quando possível; incluir `#encaminhamento`).
- **Riscos / bloqueios** (se houver).
- **Kaizens** (melhorias contínuas sugeridas, se houver).
